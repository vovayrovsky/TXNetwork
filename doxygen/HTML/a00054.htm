unsigned txMouseButtons ( )

Возвращает состояние Кнопок Мыши!

Возвращает
Состояние Кнопок Мыши! 
     В возвращаемом значении выставленный в единицу 1-й (младший) бит означает нажатую левую Кнопку Мыши,
     2-й - правую. <br>
     Например, возвращенное число 3 (двоичное 11) означает одновременное нажатие левой и правой Кнопок.
См. также
txMouseX(), txMouseY(), txMousePos(), txMouseButtons()
Примеры использования:
1 while (txMouseButtons() != 3)
2  {
3  if (txMouseButtons() & 1) txCircle (txMouseX(), txMouseY(), 20);
4  if (txMouseButtons() & 2) txLine (txMouseX(), txMouseY(), 0, 0);
5  txSleep (0);
6  }
